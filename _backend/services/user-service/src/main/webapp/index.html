User Services - API Documentation

Overview

API Root: http://localhost:9081/users

Build: Run ./build.bat to compile.

Start Service: Run mvn liberty:run to start the service. mvn liberty:dev to start the service as dev.

API Endpoints

Accounts

@GET

Retrieve user account by ID:

http://localhost:9081/users/account/id

Authentication

@GET

Retrieve authentication token:

http://localhost:9081/users/auth/token
